# Storegg

Mobile marketplace with a twist: combines buy and sell funtionality with minigames.

---

## Local Development

### Requirement

- Node.js
- Expo Go App on Play Store

### Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/itsdomibro/mobile-storegg.git
   cd mobile-storegg
   ```

2. Installing Expo:

   ```bash
   npm install expo
   ```

3. Install dependencies:

   ```bash
   npx expo install
   ```

4. Start the development server:

   ```bash
   npx expo start
   ```

5. Open Expo Go app and scan the given QR Code
